# Portfólio V2

Esta é a segunda versão do meu portfólio pessoal, desenvolvida com o objetivo de apresentar meus projetos, habilidades, experiências e evolução como desenvolvedor. O projeto funciona como um espaço centralizado para reunir minha trajetória acadêmica, conhecimentos técnicos e projetos desenvolvidos ao longo dos estudos.

## Visão Geral do Projeto

A proposta desta versão foi criar uma interface mais moderna, organizada e responsiva, oferecendo uma experiência de navegação consistente em diferentes dispositivos.

Além da reformulação visual, o projeto recebeu melhorias estruturais em relação à primeira versão, incluindo sistema de temas, melhor organização do código e uma apresentação mais profissional das informações.

O portfólio conta com seções dedicadas para:

- Apresentação pessoal
- Projetos desenvolvidos
- Sobre mim
- Currículo e habilidades
- Informações de contato

## Principais Funcionalidades

- Interface moderna e responsiva
- Alternância entre tema claro e escuro
- Tradução dinâmica entre Português e Inglês
- Exibição organizada de projetos
- Download de currículo em PDF
- Integração com GitHub e LinkedIn
- Sistema de partículas animadas em segundo plano
- Exibição dinâmica de projetos adicionais

## Tecnologias Utilizadas

### HTML5

Utilizado para estruturar todo o conteúdo do portfólio e organizar suas diferentes seções.

### CSS3

Responsável pela estilização da interface, responsividade, animações, temas e organização visual dos componentes.

### JavaScript

Responsável pelas funcionalidades dinâmicas do projeto:

- Sistema de tradução PT-BR / EN
- Alternância entre tema claro e escuro
- Exibição de projetos adicionais
- Gerenciamento dinâmico dos textos da interface
- Controle das animações de fundo
- Persistência das preferências de idioma

Todo o conteúdo textual do site é centralizado em um arquivo JavaScript de traduções, permitindo manutenção simplificada e troca dinâmica de idioma sem recarregar a página.

### Bibliotecas Utilizadas

* Lucide Icons — Ícones da interface
* Font Awesome — Ícones complementares
* tsParticles — Sistema de partículas animadas

## Melhorias Futuras

- Inclusão de novos projetos
- Aprimoramento contínuo da interface
- Expansão das animações e interações
- Integração com APIs externas
- Desenvolvimento de novas funcionalidades conforme evolução dos estudos

## Objetivo

Este projeto representa minha evolução como desenvolvedor e reúne conhecimentos adquiridos em desenvolvimento web, design de interfaces e experiência do usuário. O portfólio continuará recebendo melhorias e atualizações conforme avanço nos estudos e desenvolvo novos projetos.

# Portfolio V2

This is the second version of my personal portfolio, developed to showcase my projects, skills, experiences, and growth as a developer. The project serves as a centralized space where visitors can learn about my academic journey, technical skills, and projects developed throughout my studies.

## Project Overview

The goal of this version was to create a more modern, organized, and responsive interface, providing a consistent browsing experience across different devices.

In addition to the visual redesign, the project includes structural improvements compared to the first version, such as multilingual support, theme switching, better code organization, and a more professional presentation of information.

The portfolio includes sections dedicated to:

- Personal introduction
- Featured projects
- About me
- Resume and skills
- Contact information

## Main Features

- Modern and responsive interface
- Light and dark mode toggle
- Dynamic Portuguese and English translation
- Organized project showcase
- Resume download in PDF format
- GitHub and LinkedIn integration
- Animated particle background
- Dynamic display of additional projects

## Technologies Used

### HTML5

Used to structure all portfolio content and organize its different sections.

### CSS3

Responsible for styling, responsiveness, animations, themes, and visual organization of components.

### JavaScript

Used to control the project's dynamic features:

- PT-BR / EN translation system
- Light and dark mode toggle
- Additional project display
- Dynamic text management
- Background animation control
- Language preference persistence

All website text is centralized in a JavaScript translation file, allowing easier maintenance and dynamic language switching without reloading the page.

### Libraries Used

- Lucide Icons — Interface icons
- Font Awesome — Additional icons
- tsParticles — Animated particle system

## Future Improvements

- Addition of new projects
- Continuous UI improvements
- Expanded animations and interactions
- Integration with external APIs
- New features as my studies and skills evolve

## Purpose

This project represents my growth as a developer and brings together knowledge acquired in web development, interface design, and user experience. The portfolio will continue to evolve as I learn new technologies and develop new projects.
